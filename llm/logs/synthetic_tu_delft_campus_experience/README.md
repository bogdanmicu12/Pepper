# Synthetic TU Delft Campus Experience Transcript

This folder contains a synthetic 20-minute two-participant conversation for testing Mercer-style coding and the Pepper LLM transcript workflow.

Files:

- `transcript.csv`: raw-style transcript using the same columns as `llm/logs/transcript.csv`.
- `mercer_coding_template.csv`: a lighter coding sheet with one row per utterance and an empty `mercer_code` field.

Conversation setup:

- Session: `SYN-TUDELFT-001`
- Group: `G01`
- Conversation: `campus-experience-20min`
- Theme: TU Delft campus experience, student wellbeing, study spaces, belonging, wayfinding, food, commuting, and group work.
- Duration: 20 minutes, from `2026-05-09T14:00:00+02:00` to `2026-05-09T14:20:00+02:00`.
- Participants: `Participant 1` and `Participant 2`.
- Robot trigger word: `Pepper`.

Mercer 2005 coding reminder:

- `disputational`: disagreement or unresolved individual positions.
- `cumulative`: uncritical agreement, repetition, or additive confirmation.
- `exploratory`: critical but constructive reasoning where participants challenge, justify, and co-construct ideas.

The raw transcript intentionally leaves Mercer codes blank so the same data can be coded independently.
